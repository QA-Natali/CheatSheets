### Тип Строка

Для того, чтобы создать переменную типа строка,  
её значение нужно взять в одинарныее или двойные кавычки.  
```py
"Привет, мир!"  
'Привет, мир!'  
```
Строки можно складывать с помощью оператора '+' и если строки  
не содержат переменных, то их можно складыать без оператора '+'.  
```py

'Привет,' 'мир!'  
```

### Тип кортежи (tuple)  

Кортежи похожи на списки, которые нельзя изменять.  

```py
tup = (1, 2, 3)
``` 

Все операции над списками, не изменяющие список  
(сложение, умножение на число, методы index() и count() и некоторые другие операции).  
Можно также по-разному менять элементы местами и так далее.  

Важно помнить, что для создания кортежа из одного элемента, после него  
нужно поставить запятую, в любом другом случае это не обязательно делать.  

```py
tup = ('s', )

# или даже так  

tup = 's',
```
**Операции меняющие множества**  

| Метод | Что делает |
|-------|------------|
| set.update(other, ...); set \|= other \| | объединение |
| set.intersection_update(other, ...); set &= other & | пересечение |

| set.symmetric_difference_update(other); set ^= other | множество из элементов, встречающихся в одном множестве, но не встречающиеся в обоих |

| set.remove(elem) | удаляет элемент из множества. KeyError, если такого элемента не существует |

| set.pop() | удаляет первый элемент из множества. Так как множества не упорядочены, нельзя точно сказать, какой элемент будет первым |
| set.clear() | очистка множества |

## Тип Словари

Словари в Python - неупорядоченные коллекции произвольных объектов с доступом по ключу.  
Их иногда ещё называют ассоциативными массивами или хеш-таблицами.  

**Как создать словарь:**  

Объявляем переменную, в которой хранится словарь 

```py
filled_dict = {'dict': 1, 'dictionary': 2}

empty_dict = {}   # Пустой словарь  
```

С помощью функции dict:

```py
filled_dict = dict(short='dict', long='dictionary')

number_dict = dict([(1, 1), (2, 4)])
```

С помощью метода fromkeys:

```py
filled_dict = dict.fromkeys(['a', 'b']) # {'a': None, 'b': None}

yet_dict = dict.fromkeys(['a', 'b'], 100) # {'a': 100, 'b': 100}
```

C помощью генераторов словарей  

```py
filled_dict = {a: a ** 2 for a in range(7)} # {0: 0, 1: 1, 2: 4, 3: 9, 4: 16, 5: 25, 6: 36}
```

**Методы работы со словарём**

|                 Метод                 | Что делает |
|---------------------------------------|------------|
|dict.clear()                           |Очищает словарь|
|dict.copy()                            |Возвращает копию словаря|
|classmethod dict.fromkeys(seq[, value])|Создает словарь с ключами из seq и значением value (по умолчанию None)|
|dict.get(key[, default])               |Возвращает значение ключа, но если его нет, не бросает исключение, а возвращает default (по умолчанию None)|
|dict.items()                           |Возвращает пары (ключ, значение)|
|dict.keys()                            |Возвращает ключи в словаре|
|dict.pop(key[, default])               |Удаляет ключ и возвращает значение. Если ключа нет, возвращает default (по умолчанию бросает исключение)|
|dict.popitem()                         |Удаляет и возвращает пару (ключ, значение). Если словарь пуст, бросает исключение KeyError. Помните, что словари неупорядочены|
|dict.setdefault(key[, default])        |Возвращает значение ключа, но если его нет, не бросает исключение, а создает ключ со значением default (по умолчанию None)|
|dict.update([other])                   |Обновляет словарь, добавляя пары (ключ, значение) из other. Существующие ключи перезаписываются. Возвращает None (не новый словарь!)|
|dict.values()                          |Возвращает значения в словаре|

## Условные операторы и циклы  

```py
variable = 5

if variables > 2:
    pass # Место для инструкции 1
    pass # Место для инструкции 2
    pass
    pass #  Место для инструкции n
```
Если условие соблюдается, то мы попадаем в тело условного оператора if  
и начинаем выполнять все инструкции по порядку.  
Если условие не соблюдается, то мы не выполняем инструкции  теле оператора if.  

```py
variable = 5

if variables > 40:
    pass # Место для инструкции 1
    pass # Место для инструкции 2
    pass
    pass # Место для инструкции n
else:
    pass # Мусто для инструкции 1
    pass # Мусто для инструкции 2
    pass
    pass # Место для инструкции n
```
### Условные операторы if, elif и else  

Когда дного if мало, то на помощь приходит оператор elif, он записыается  
ровно также  как и if. Таких операторов в вашей программе может быть сколько  
угодно много. Он всегда идет вторым после if.  

```py
variable = 5

if variables > 40:
    pass # Место для инструкции 1
    pass # Место для инструкции 2
    pass
    pass # Место для инструкции n
elif variables > 0:
    pass # Место для инструкции 1
    pass # Место для инструкции 2
    pass
    pass # Место для инструкции n
else:
    pass # Место для инструкции 1
    pass # Место для инструкции 2
    pass
    pass # Место для инструкции n
```

## Цикл For  

Цикл For служит для того чтобы легко сделать проход по списку.  
Если вы знаете сколько итераций вам нужно сделать, но списка у вас нет,  
то его можно создать с помощью оператора `range`.

```py
item_list = ["молоко", "хлеб", "колбасу", "икру красную", "икру чёрную", "икру кабачковую заморскую"]
for i in item_list:
    # Место для инструкций  
    print("Купи обязательно " + i)     # Пример инструкции
    # Место для инструкций
```
Оператор range([начало], конец , [шаг]) возвращает список чисел от начала до конца с указанным шагом,  
по умолчанию шаг равен 1, а начало равно 0.  
Конец - является обязательным параметром. Если указать два параметра, то в оператор передадутся знанчения  
начала и конца списка, а для указания шага, нужно передать все три параметра.  

Например:
```py
for i in range(4):
    print(i)
"""
# Выводит:
0
1
2
3
"""

for i in range(4,8)
print(i)
"""
# Выодит:
4
5
6
7
"""
for i in range(4, 8, 2):
    print(i)
"""
# Выводит:
    4
    6
"""
```

### Цикл While

Циклы While работают до тех пор, пока входное условие не станет ложным.  
Нужно быть осторожным, чтобы не попасть в вченый цикл, если вы этого не хотите. 

```py
x = 0
while x < 5:
    print(x)
    x = x + 1
"""
# Выводит:
    0
    1
    2
    3
    4
"""
```
